휴대폰 인증

본인 확인을 위해 휴대폰 인증을 진행해주세요.

항목 안내
①
이름 — 신청자 본인의 실명을 입력합니다. 허위 기재 시 신청이 거절될 수 있습니다.
②
휴대폰 번호 — 본인 명의의 휴대폰 번호를 입력합니다. 형식: 010-XXXX-XXXX. 숫자를 입력하면 자동으로 하이픈이 삽입됩니다.
주의 타인 명의 번호는 사용할 수 없습니다.
③
다음 버튼 — 번호 유효성 확인 후 Step 2(사업자 인증)로 이동합니다. 올바르지 않은 번호 형식이면 오류 메시지가 표시됩니다.
2
사업자 인증
사업자등록증 기준 정보를 입력합니다 (홈택스 정보와 일치해야 합니다)

사업자 정보

몰 신청 전 사업자 정보를 인증해주세요.

항목 안내
①
상호명 — 사업자등록증에 기재된 상호명을 그대로 입력합니다. 예: (주)이지샵
②
사업자등록번호 — 숫자 10자리. 입력 시 000-00-00000 형식으로 자동 변환됩니다.
③
대표자명 — 사업자등록증상의 대표자 이름을 입력합니다.
④
개업연월일 — 사업자등록증의 개업일. 입력 시 YYYY.MM.DD 형식으로 자동 변환됩니다.
주의 입력 정보가 홈택스(국세청) 데이터와 일치하지 않으면 인증이 거절됩니다.
⑤
다음 버튼 — 필수 항목 4개를 모두 입력해야 활성화됩니다. 클릭 후 Step 3(정보 입력)으로 이동합니다.

항목 안내
①
몰 유형 선택
· 일반몰: 고객용 쇼핑몰 페이지 + 회원/리뷰/문의/테마/배너/약관 등 전체 메뉴 사용 가능
· 입점몰 (운영 전용): 고객용 페이지 없이 상품·주문·배송·통계·설정 메뉴만 사용. 통합몰에 입점해 운영할 때 선택
②
약관 동의 — 모든 필수 약관에 체크해야 아래 쇼핑몰 정보 폼이 활성화됩니다.
③
전체 동의 — 체크 한 번으로 모든 필수 약관에 일괄 동의합니다. 각 약관 우측 내용 보기 ▾를 클릭하면 전문이 펼쳐집니다.
④
입점몰 전용 약관 — 몰 유형을 입점몰로 선택한 경우에만 표시됩니다.

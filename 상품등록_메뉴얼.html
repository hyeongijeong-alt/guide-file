기본 비노출 — 체크 시 온마켓에 함께 노출됩니다.

항목 안내
①
카테고리 필수 — 1차 선택 시 2차 드롭다운이 자동으로 활성화됩니다. 하위 카테고리가 있는 경우 최하위까지 선택해야 저장이 가능합니다.
선택 가능한 1차 카테고리: 가구/인테리어, 도서, 디지털/가전, 생활/건강, 스포츠/레저, 식품, 여가/생활편의, 출산/육아, 패션의류, 패션잡화, 화장품/미용
②
상품명 필수 — 고객에게 노출되는 대표 상품명입니다. 최대 200자 이내로 입력합니다.
③
판매 상태 — 기본값은 판매중입니다.
· 대기 준비 중 상품 / 검수 전   · 판매중 즉시 판매   · 판매중지 일시 중단   · 품절 재고 소진 (프론트 노출, 구매 불가)
④
온마켓에 노출 선택 — 체크 시 온마켓(통합 채널)에도 상품이 노출됩니다. 자사몰 전용으로만 판매하려면 체크하지 않습니다.

항목 안내
①
옵션 섹션 — 옵션을 추가하지 않으면 단일 상품으로 저장됩니다. 옵션이 불필요한 경우 건너뜁니다.
②
+ 옵션 그룹 추가 — 클릭할 때마다 새 옵션 그룹(색상, 사이즈 등)이 추가됩니다. 그룹은 여러 개 추가할 수 있습니다.
③
그룹명 입력 — 옵션의 종류를 입력합니다. 예: 색상, 사이즈, 용량
④
옵션 값 입력 — 각 그룹에 옵션 값을 입력합니다. 예: 색상 그룹 → 블랙, 화이트, 네이비. + 옵션 값 추가로 행을 늘립니다.
⑤
자동 생성 조합 테이블 — 그룹·값을 입력하면 모든 조합이 자동 생성됩니다 (예: 3색상 × 3사이즈 = 9조합).
· 추가 가격: 판매가에 더해지는 금액. 0원이면 동일 가격.
· 재고: 조합별 재고 수량 (0이면 품절 처리).
· 판매 체크박스: 체크 해제 시 해당 조합 판매 중지.
일괄 적용 가격·재고를 일괄 입력 후 적용 버튼으로 전체 조합에 동시 설정할 수 있습니다.
⑥
옵션 전체 삭제 주의 — 클릭 시 확인 창이 뜨며, 확인하면 모든 그룹과 조합이 삭제되고 단일 상품으로 전환됩니다.

유효성 오류 — 필수 항목 누락 또는 가격 규칙 위반 시 오류 메시지가 해당 필드 아래에 표시됩니다. 오류가 있는 섹션으로 자동 스크롤됩니다.
저장 전 체크리스트:
  • 카테고리 — 최하위까지 선택 완료
